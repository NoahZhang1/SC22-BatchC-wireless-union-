HOW TO USE?







The Lorem Ipsum bot is a text generation bot that acts as a writing assistant for the user. The user will be able to select from a drop-down of different scenarios and the bot will auto-fill certain sentences based on the scenario given








TECHNOLOGY STACK







We used a GPT-2 aitextgen model for headline generation. All models were trained by finetuning the 124M pre-trained language model. We also used other models for text summarization and grammar correction.








DATASET







We split up into three different subgroups to work on training three different models: Human Conversation, Customer Support, and Academic Conversation. Each of these groups trained similar models with different online datasets








USE CASES AND EXPANSION







This text generation project can be used in an assortment of problems, such as conversations. You can use this in any conversation apps, it can generate commonly used text to complete your sentence. It also has a grammar corrector built in.








CHALLENGES WE FACED







We had some challenges along the way. Finding datasets was one of the first issues that we ran into. There weren't many datasets that matched what we were looking for. Preprocessing the data was also a challenge. Almost all of the datasets we found were formatted unorthodox ways. Another challenge was training the AI model. This took a very long time due to webpage issues and large dataset files, this slowed us down a decent amount. We also had some issues with Google Colaboratory, such as us having to reload the files and code blocks every time we reopened the project.
